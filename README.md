# 👑 N-Queen Solver

## Project Description

The **N-Queen Solver** is a problem-solving application designed to solve the famous **N-Queens Puzzle** using algorithmic techniques and backtracking.

The goal of the puzzle is to place **N queens** on an **N × N chessboard** so that no two queens attack each other. This means:

* No two queens can be in the same row.
* No two queens can be in the same column.
* No two queens can be on the same diagonal.

This project demonstrates the use of recursion, backtracking algorithms, and logical problem-solving techniques in programming.

The application allows users to:

* Enter the board size (N).
* Generate valid queen arrangements.
* Visualize the solution on the chessboard.
* Understand how backtracking works in solving complex problems.

# How the Algorithm Works

The solver uses the **Backtracking Algorithm**:

1. Place a queen in a safe position.
2. Move to the next row.
3. Continue placing queens safely.
4. If no safe position exists, backtrack to the previous queen.
5. Repeat until all queens are placed successfully.

This approach ensures all possible valid solutions are explored efficiently.
